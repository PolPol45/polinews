# Calendar - 4 Week Pilot

## Cadence
- Day 1 kickoff
- Checkpoint every 2 days
- Weekly checkpoint on days 7, 14, 21, 28

## Fixed slots (local time)
- 09:00 ingestion check
- 11:00 KPI drift check
- 14:00 fraud review
- 17:00 budget/redeem check
- 19:00 daily report

## Checkpoint agenda (30 min)
1. Completed tasks vs planned tasks
2. Blockers and owner actions
3. KPI trend and anomalies
4. Risk review
5. Next 48h priorities

## Mandatory outputs per checkpoint
- Updated task status
- Updated blocker list
- Evidence links collected
- Next checkpoint goals
