# Board Template - MVP Off-Chain

Columns:
1. Backlog
2. In progress
3. Blocked
4. Done

Card format:
- Title: `<TASK_ID> <short task title>`
- Body:
  - Owner:
  - Priority:
  - Depends on:
  - DoD:
  - Evidence path:
  - Due day:

Minimum board rules:
- One owner per card.
- A card cannot move to `Done` without evidence link.
- `Blocked` cards must include blocker reason and unblock owner.
- WIP limit: max 3 `In progress` cards for small team.

Startup checklist:
- Create all cards from `MASTER_CHECKLIST.md`.
- Tag P0 cards in red.
- Pin weekly checkpoint cards `W1-12`, `W2-13`, `W3-12`, `W4-10`.
