# incident_response_matrix

| Severity | Description | Action | Owner | Communication |
|---|---|---|---|---|
| Critical | Core flow unavailable or financial risk | Freeze risky path and mitigate immediately | Founder + Freelance | Immediate checkpoint update |
| High | Major degradation with workaround | Apply workaround and fix within SLA | Assigned owner | Same-day update |
| Medium | Partial issue | Plan fix in next cycle | Assigned owner | Daily report |
| Low | Minor issue | Backlog | Assigned owner | Weekly review |
