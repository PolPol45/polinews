# attribution_policy_v1

Mandatory elements on each Poli-News story page:
- Original source name.
- Outbound link to source article.
- Publish timestamp (source or ingestion fallback).
- Label: "Summary generated by Poli-News from public source metadata".

Prohibitions:
- No full-text copy of source article.
- No paywalled content reproduction.

UI checks:
- Source block must appear above quiz box.
- At least one source link must be visible without expansion.
