## Mandatory elements on each Poli-News story page

- Label: **"Summary generated by Poli-News"** — displayed above the source links.
- Source links: one or more outbound links to the original source articles, visible directly below the label without any expansion or click required.
- Publish timestamp: source timestamp or ingestion fallback if missing.

## Prohibitions

- No full-text copy of source article.
- No paywalled content reproduction.
- Do not add additional legal disclaimers or technical explanations to the label — the approved text is exactly: `Summary generated by Poli-News`.

## UI placement rules

- Label + source links block must appear **above** the quiz box — always.
- At least one source link must be visible without expansion or extra interaction.
- Source link format: `[Source name] → [outbound URL]`
