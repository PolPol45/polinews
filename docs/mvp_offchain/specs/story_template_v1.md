# story_template_v1

## Story page structure
1. Topic badge + headline
2. Short snippet (from feed metadata)
3. Key points (3-5 bullets, internal synthesis)
4. Sources list (name + link)
5. Verification box (quiz/comment)
6. Credits balance teaser

## Content guardrails
- Keep snippet under 320 chars.
- Key points must be non-duplicative.
- No claims without source anchor.
- Do not insert unverifiable facts.

## Metadata
- `story_id`
- `topic_slug`
- `source_count`
- `published_at`
- `generated_at`
